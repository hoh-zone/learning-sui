# 第十二章 · 测试

本章系统讲解 Sui Move 的测试体系，从单元测试到多角色场景测试，从覆盖率到 Gas 分析，帮助你编写高质量的测试代码。

## 本章内容

| 节 | 主题 | 你将学到 |
|---|------|---------|
| 12.1 | 测试基础 | #[test]、#[expected_failure]、运行命令 |
| 12.2 | 好的测试 | 覆盖正常/异常路径、命名规范 |
| 12.3 | 测试工具函数 | #[test_only] 模块、辅助函数 |
| 12.4 | 模拟交易上下文 | dummy()、自定义 sender |
| 12.5 | 测试场景 | 多角色多交易、take / return |
| 12.6 | 使用系统对象 | Clock、Random 的测试模拟 |
| 12.7 | Builder 模式测试 | 链式构造复杂测试数据 |
| 12.8 | 随机输入测试 | 模糊测试、随机用例生成 |
| 12.9 | 扩展外部模块 | 测试第三方依赖 |
| 12.10 | 覆盖率报告 | 生成和解读覆盖率数据 |
| 12.11 | Gas Profiling | Gas 消耗分析与优化 |

## 学习目标

读完本章后，你将能够：

- 为合约编写全面的单元测试和场景测试
- 使用 Test Scenario 模拟多用户交互
- 分析测试覆盖率和 Gas 消耗
