# 第五章 · Move 语言基础

本章系统讲解 Move 语言的全部语法特性，从类型系统到函数，从所有权到泛型，为后续的 Sui 开发打下扎实的语言基础。

## 学习目标

读完本章后，你将能够：

- 熟练使用 Move 的所有基础语法
- 理解 Move 独有的所有权和能力系统
- 编写包含泛型、枚举和模式匹配的 Move 代码
