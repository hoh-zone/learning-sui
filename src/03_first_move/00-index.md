# 第三章 · 第一个 Move 程序

本章将带你动手编写、编译、测试和部署第一个 Move 智能合约，完成从代码到链上的全流程。本章属于**偏实战的入门综合章**：在尚未讲完语言细节的前提下，先让你跑通工具链与发布流程，后续第四至七章再系统补全语法与语义。

## 本章内容

| 节 | 主题 | 你将学到 |
|---|------|---------|
| 3.1 | Hello World | 创建项目、`Hello` 对象、`entry` 铸造并转移给自己、编译与测试 |
| 3.2 | Hello Sui | 将合约部署到 devnet、理解发布输出 |
| 3.3 | 与合约交互 | 用 CLI 调用合约函数、查看对象和交易 |

## 学习目标

读完本章后，你将能够：

- 独立创建一个 Move 项目并编写简单模块
- 将合约部署到 Sui devnet
- 通过 CLI 调用链上合约并查看结果
