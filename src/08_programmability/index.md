# 第十章 · 高级可编程性

本章介绍 Sui Framework 提供的高级编程能力，包括动态字段、集合、代币操作、密码学等，让你的合约具备丰富的链上功能。

## 本章内容

| 节 | 主题 | 你将学到 |
|---|------|---------|
| 10.1 | Sui Framework 概览 | 框架结构、核心模块 |
| 10.2 | 交易上下文 | TxContext 的方法和传递规则 |
| 10.3 | 模块初始化器 | init 函数的签名要求和执行时机 |
| 10.4 | 事件 | 事件定义、emit、链下监听 |
| 10.5 | Epoch 与时间 | 基于时间的逻辑 |
| 10.6 | 集合类型 | VecMap / VecSet 的使用 |
| 10.7 | 动态字段 | 异构存储、增删改查 |
| 10.8 | 动态对象字段 | 与动态字段的区别、链上可查询 |
| 10.9 | 派生对象（derived_object） | 确定性地址、claim/exists、注册表模式 |
| 10.10 | 动态集合 | Table / Bag / ObjectTable / ObjectBag |
| 10.11 | Balance 与 Coin | 代币的底层操作 |
| 10.12 | BCS 序列化 | 编码 / 解码、链下参数构造 |
| 10.13 | 密码学与哈希 | SHA / ED25519 / ECDSA |
| 10.14 | 链上随机数 | Random 对象、公平性保证 |

## 学习目标

读完本章后，你将能够：

- 使用动态字段与动态对象字段实现灵活的数据存储
- 使用派生对象（derived_object）实现确定性地址与注册表模式
- 操作 Balance 和 Coin 实现代币逻辑
- 在合约中使用密码学原语和链上随机数
