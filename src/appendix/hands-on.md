# 附录 · 实战练习

## 实战一：CLI 自检

1. 运行 `src/appendix/code/cli_demo/demo.sh`。
2. 对照附录 D《CLI 速查表》，执行一条你平时最少用的子命令（如 `sui client gas` / `sui client verify-bytecode-meter` 等，以本机 CLI 为准）。
3. **验收**：记录命令输出中与「网络 / 地址 / gas」相关的字段。

## 实战二：Transfer 函数迷宫

1. 打开附录 C《Transfer 函数参考》。
2. 针对「共享对象 + 仅允许特定模块调用」的场景，写出应优先考虑的 API 名称（不必上链）。
3. **验收**：2～3 句理由。

## 实战三：术语表自测

1. 从附录 A 随机抽 **5 个**术语，遮住定义自测释义。
2. 对照原文订正。
3. **验收**：自测正确率写入笔记（目标 ≥ 3/5）。
