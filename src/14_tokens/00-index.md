# 第十四章 · 代币经济

本章讲解如何在 Sui 上创建和管理各种类型的代币，从标准 Coin 到受监管代币再到闭环 Token。

## 本章内容

| 节 | 主题 | 你将学到 |
|---|------|---------|
| 14.1 | 创建自定义 Coin | OTW + coin_registry（new_currency_with_otw / finalize）、铸造与销毁 |
| 14.2 | Coin 元数据 | 名称、符号、精度、图标 |
| 14.3 | Treasury Cap 管理 | 固定供应 vs 无限供应、锁定策略 |
| 14.4 | 受监管代币 | DenyCap、黑名单、合规设计 |
| 14.5 | 闭环代币 | Token vs Coin、TokenPolicy、消费规则 |
| 14.6 | 游戏内代币实战 | 积分系统、双币经济、代币兑换 |

## 学习目标

读完本章后，你将能够：

- 创建并发布自己的代币
- 实现固定供应和受监管的代币逻辑
- 设计游戏内的代币经济系统
