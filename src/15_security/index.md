# 第十八章 · 安全与升级

本章讲解 Sui Move 合约的安全最佳实践，帮助你构建生产级别的安全合约。包升级的完整内容已独立成第十五章。

## 本章内容

| 节 | 主题 | 你将学到 |
|---|------|---------|
| 18.1 | 安全最佳实践 | 权限检查、输入验证、对象安全 |
| 18.2 | 常见漏洞模式 | 权限泄露、对象混淆、整数溢出 |
| 18.3 | 错误处理最佳实践 | 错误码设计、分类、用户友好的错误信息 |
| 18.4 | 协议与网络限制 | 交易/对象/参数/动态字段/事件等协议上限 |

## 学习目标

读完本章后，你将能够：

- 识别和防范常见的安全漏洞
- 设计健壮的错误处理策略
- 在合约中落实权限与输入验证等安全实践
